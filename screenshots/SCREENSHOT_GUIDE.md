# Screenshot Guide — FurEver AI Concierge

This folder is used to store screenshots for the capstone submission.

Recommended screenshots:

1. Home / app overview
   - Shows the FurEver AI Concierge title
   - Shows course concepts listed in the app
   - Shows the multi-agent workflow summary

2. Add Memory tab
   - Shows the Memory Agent, Evaluation Agent, and Decision Router explanation
   - Optional: show a safe memory being processed

3. Timeline tab
   - Shows approved RuRu memories
   - Shows that only approved memories enter the timeline

4. Human Review tab
   - Shows a sensitive memory paused for review
   - Shows editable fields
   - Shows Evaluation Agent risk flags
   - Shows human-in-the-loop review

5. Life Story tab
   - Shows Story Agent explanation
   - Shows generated RuRu story from approved memories

6. Sitter Share tab
   - Shows Safety Agent explanation
   - Shows sitter-safe care card
   - Shows privacy-aware sharing

7. Observability tab
   - Shows local trace log
   - Shows agent steps such as memory_agent, evaluation_agent, decision_router, story_agent, and safety_agent

8. Demo Controls tab
   - Shows reset/load demo data controls
